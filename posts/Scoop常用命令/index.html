Scoop常用命令

发表于 2026-07-13 分类于 生活

Scoop是一款命令行包管理器，旨在便捷管理

Scoop介绍

Scoop是一款为Windows系统设计的命令行包管理器。它通过命令行完成软件的安装、更新、卸载和版本管理。
关于Scoop的安装，可以看这个网站Scoop

flowchart TD
    A[执行 scoop install git]
    B[读取 Bucket 中的 Manifest]
    C[下载软件]
    D[校验 SHA256]
    E[解压安装]
    F[创建 current 软链接]
    G[配置 PATH]
    H[安装完成]

    A --> B
    B --> C
    C --> D
    D --> E
    E --> F
    F --> G
    G --> H
scoop安装软件的过程

以前，我们是这样安装软件的：

flowchart LR
	A[打开浏览器]
	B[搜索软件名称，打开官网]
	C[官网下载软件安装包或者便携版本]
	D[通过安装包安装软件]
	A ---> B 
	B ---> C
	C ---> D

现在，使用scoop安装软件：

flowchart LR
	A[打开命令行工具PowerShell]
	B[进入scoop网站搜索软件名称]
	C[选择仓库bucket]
	D[通过命令行工具下载软件]
	A ---> B
	B ---> C
	C ---> D

注意，不是所有的软件都推荐使用scoop安装！！！

推荐使用scoop安装的软件 不推荐使用scoop安装的软件
Git、Python、Node.js等更新频繁、经常需要配置环境变量、命令行使用较多、不依赖复杂的系统服务的软件 大型商业软件（Adobe全家桶、CAD、Unity）和集成开发环境（Visual Studio、Rider）
一些开源软件（OBS） 杀毒软件、安全软件（卡巴斯基、火绒）
轻量编辑器（VS Code、Sublime Text） 硬件驱动软件（AMD Driver）、外设驱动软件（G HUB）
小型工具（Everything、Privazer） 国产大型软件（如百度网盘等强制自动更新的软件）
命令行工具 游戏平台（Steam、Epic）

Scoop常用命令

命令 功能
scoop install <软件名>
例如：scoop install git		 安装软件
scoop uninstall <软件名>
例如：scoop uninstall git		 卸载软件
scoop update <软件名>
例如：scoop update git		 更新软件
scoop update 更新scoop
scoop info <软件名>
例如：scoop info git		 查看软件信息
scoop list 查看已安装软件列表
scoop bucket list 查看仓库列表
scoop bucket add <仓库名>
例如：scoop bucket add extras		 添加仓库
scoop bucket rm <仓库名>
例如：scoop bucket rm extras		 移除仓库
scoop cache rm <软件名>
例如：scoop cache rm WeChat		 删除缓存
scoop cleanup <软件名>
例如：scoop cleanup git		 删除软件旧版本
scoop export > <自定义文件名称>.json 导出软件列表
scoop import > <自定义文件名称>.json 导入软件列表
scoop reset <软件名>
例如：scoop reset git		 重置软件软链接（Current文件）
scoop help 查看更多命令
